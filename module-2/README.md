### Module 2: ถอดรหัสลับ
จากข้อที่ `Module 1: เข้ารหัสลับ` จงเขียนโปรแกรมถอดรหัสดังกล่าว

### Requirement
แก้ logic ฟังก์ชั่น `input_func(input)` ที่รับ input เป็นข้อความที่เข้ารหัสแล้ว return ออกไปเป็นข้อความก่อนที่จะทำการเข้ารหัส

```python
aa = a
ab = b
ac = c
ad = d
ae = e
af = f
ag = g
ah = h
ai = i
aj = j
ba = k
cf = z
```

### Example Input
```python
aaabacad
```

### Example Output
```
abcd
```
